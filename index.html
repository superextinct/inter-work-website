Inter Work is a variable typeface based on the Inter font family.

Inter Work was crafted as a corporate font for an initiative to transform vacant retail premises into Co-Working & Creative Spaces. Just like those spaces adapt to the available area, the typeface should as well. Inter Work is an uppercase font that is to be used alongside its predecessor Inter.

You are free to use this font in almost any way imaginable. Inter Work is a free and open-source font-family, licensed under SIL Open Font License (1.1). For extended commercial use, please get in touch.

Weights & Styles

Inter Work has been crafted with flexibility in mind. There are six weights, each with a stretched counterpart, making a total of 12 Styles.

It is offered as both traditional constant font files as well as a Variable Font which contains all styles in a much smaller file size.
